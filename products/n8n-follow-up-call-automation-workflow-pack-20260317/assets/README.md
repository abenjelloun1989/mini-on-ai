# Follow-Up Call Automation Workflow Pack

A ready-to-deploy n8n workflow that automatically logs unanswered calls, schedules follow-up reminders, and tracks callback completion for small business owners managing high call volumes.

## What's included

- Ready-to-import n8n workflow (`workflow.json`)
- 10 connected nodes
- Setup instructions for credentials and configuration

## How to import

1. Open your n8n instance
2. Go to **Workflows** → **Import from file**
3. Select `workflow.json`
4. Follow the setup steps below

## Setup

1. Step 1: Import this JSON into your n8n instance via Settings > Import Workflow.
2. Step 2: Create a Google Sheets OAuth2 credential in n8n (Credentials > New > Google Sheets OAuth2 API) and replace 'YOUR_GOOGLE_SHEETS_CREDENTIAL_ID' in both Google Sheets nodes.
3. Step 3: Create a Google Sheet named 'MissedCalls' with columns: CallerNumber, CallerName, CallTimestamp, FollowUpDeadline, CallbackStatus — then replace 'YOUR_GOOGLE_SHEET_ID' with the actual Google Sheet ID (found in the sheet URL).
4. Step 4: Create a Slack API credential in n8n and replace 'YOUR_SLACK_CREDENTIAL_ID'. Ensure the Slack bot has permission to post to your chosen channel (default: #missed-calls). Update the channel name if needed.
5. Step 5: Create an SMTP credential in n8n (Credentials > New > SMTP) with your email provider details. Replace 'YOUR_SMTP_CREDENTIAL_ID', and update 'your-email@example.com' (From) and 'owner@yourbusiness.com' (To) with real addresses.
6. Step 6: Create a Telegram Bot via BotFather (https://t.me/botfather), get the bot token, create a Telegram API credential in n8n, and replace 'YOUR_TELEGRAM_CREDENTIAL_ID'. Also replace 'YOUR_TELEGRAM_CHAT_ID' with your personal or group chat ID.
7. Step 7: Configure your phone system or VoIP provider (e.g., Twilio, RingCentral, Aircall) to POST call data to the webhook URL generated by the 'Incoming Call Webhook' node. Activate the workflow first to get the webhook URL. Expected POST body fields: caller_number, caller_name, call_status (use 'missed' for missed calls).
8. Step 8: Adjust the follow-up deadline window in the 'Parse Call Data' node — currently set to 2 hours ($now.plus({ hours: 2 })). Change to match your business SLA (e.g., 1 hour, 4 hours).
9. Step 9: The 'Schedule Reminder Check' node runs every 30 minutes by default. Adjust the interval in its parameters to suit your preferred check frequency.
10. Step 10: To mark a call as completed, update the 'CallbackStatus' column in your Google Sheet from 'pending' to 'completed' after the callback is made. Consider adding a second webhook endpoint or Google Apps Script button to automate this update.

## Requirements

- n8n (self-hosted or n8n.cloud)
- Relevant credentials configured in n8n (see setup steps above)
