# Claude Code HTML Presentation Prompt Pack

> This trending skill proves massive demand for AI-generated HTML slide decks, but there's no prompt pack teaching freelancers/devs how to prompt their way to polished presentations without coding. The gap is a structured pack of 25 prompts covering layouts, themes, content types, and client deliverables.

---

## Prompts (20 total)

### 1. Full Slide Deck From Scratch

**Use case:** Starting a brand-new client presentation from zero

```
Create a complete, self-contained HTML presentation for [TOPIC] targeting [TARGET AUDIENCE]. The deck should have exactly 10 slides: (1) title slide, (2) agenda, (3-8) content slides, (9) key takeaways, (10) call-to-action/contact slide. Use a modern, professional design with a color palette of [PRIMARY COLOR] and [ACCENT COLOR]. Include smooth CSS slide transitions, a clickable navigation bar with slide numbers, keyboard arrow-key navigation, and a progress bar at the top. Each content slide must have a bold headline, 3–5 bullet points or a visual layout, and subtle entrance animations. Output a single HTML file with all CSS and JavaScript embedded. No external libraries or CDN links — everything must be inline.
```
