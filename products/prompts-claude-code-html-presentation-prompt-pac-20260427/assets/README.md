# Claude Code HTML Presentation Prompt Pack

This trending skill proves massive demand for AI-generated HTML slide decks, but there's no prompt pack teaching freelancers/devs how to prompt their way to polished presentations without coding. The gap is a structured pack of 25 prompts covering layouts, themes, content types, and client deliverables.

## What's included

- **20 prompts** ready to use with any AI assistant
- Plain text format — works with ChatGPT, Claude, Gemini, and others
- Organized by use case for easy browsing

## How to use

1. Open `prompts.md` or `prompts.json`
2. Find the prompt that matches your need
3. Copy it into your AI assistant
4. Replace any `[PLACEHOLDER]` text with your specifics
5. Run and iterate

## Files

- `prompts.md` — Human-readable prompt list
- `prompts.json` — Machine-readable structured data
- `README.md` — This file
