# n8n Carpet Photography Location Automation

A ready-to-use n8n workflow that automates the process of matching photographed carpets to suitable real-world locations using AI vision analysis, eliminating manual Gemini + Photoshop work.

## What's included

- Ready-to-import n8n workflow (`workflow.json`)
- 9 connected nodes
- Setup instructions for credentials and configuration

## How to import

1. Open your n8n instance
2. Go to **Workflows** → **Import from file**
3. Select `workflow.json`
4. Follow the setup steps below

## Setup

1. Step 1: Import this workflow JSON into n8n by going to Workflows > Import from File/URL and pasting this JSON.
2. Step 2: Obtain a Google Gemini API key from https://makersuite.google.com/app/apikey and replace 'YOUR_GEMINI_API_KEY' in the 'Gemini Vision Analysis' node URL parameter.
3. Step 3: Set up Google Sheets OAuth2 credentials in n8n (Settings > Credentials > New > Google Sheets OAuth2) and replace 'YOUR_GOOGLE_SHEETS_CREDENTIAL_ID' with the actual credential ID.
4. Step 4: Create a Google Sheet with columns: Carpet ID, Carpet Name, Submitted By, Upload Timestamp, Color Palette, Pattern Style, Texture, Mood, Ideal Locations, Room Types, Complementary Decor, Full Analysis, Processed At. Then replace 'YOUR_GOOGLE_SHEET_ID' in the 'Save to Google Sheets' node with your actual Google Sheet ID (found in the sheet URL).
5. Step 5: Create a Telegram Bot via @BotFather on Telegram, get the Bot Token, and set up Telegram credentials in n8n. Replace 'YOUR_TELEGRAM_CREDENTIAL_ID' with the credential ID and 'YOUR_TELEGRAM_CHAT_ID' with your actual Telegram chat or channel ID.
6. Step 6: Configure SMTP email credentials in n8n (Settings > Credentials > New > SMTP) for error notifications. Replace 'YOUR_SMTP_CREDENTIAL_ID', 'your-email@example.com' (sender), and 'admin@example.com' (recipient) with real values.
7. Step 7: Note the Webhook URL generated by n8n for the 'Webhook Trigger' node (format: https://your-n8n-instance.com/webhook/carpet-photo-upload). This is the endpoint your application will POST carpet images to.
8. Step 8: Configure your client application to send POST requests to the webhook URL with a JSON body containing: { 'image': '<base64_encoded_image>', 'carpet_id': '<unique_id>', 'carpet_name': '<carpet_name>', 'submitted_by': '<user_identifier>' }
9. Step 9: Ensure the base64 image sent is a JPEG or PNG format. For large images, consider compressing them before encoding to stay within Gemini API limits (max ~4MB per image).
10. Step 10: Activate the workflow by toggling the 'Active' switch in the top-right corner of the n8n workflow editor. Test by sending a sample POST request with a carpet image to the webhook URL and verify the Google Sheet is populated and Telegram notification is received.

## Requirements

- n8n (self-hosted or n8n.cloud)
- Relevant credentials configured in n8n (see setup steps above)
