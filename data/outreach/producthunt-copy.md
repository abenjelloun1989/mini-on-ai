## Description (for PH listing body)

mini-on-ai is a showcase of AI-generated digital products: Claude Code skill packs, n8n workflow templates, prompt packs, checklists, and mini-guides for engineers, freelancers, and automation builders.

**57 products available** — 15 free, 42 paid ($5–$49). New products drop weekly, generated by a Python pipeline running on a Mac mini.

- Anyone who wants a custom AI product without building it themselves
